# BLPDiffer

Compares two BLPs and produces a bitmap highlighting the differences between them.  
See the `BLPDiffer.Tests` for an example.

Supported comparison algothrims are:
- Exact pixel matching
- CIE76
- CIE94
- CIEDE2000
